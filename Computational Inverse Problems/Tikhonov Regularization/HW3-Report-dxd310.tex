{\Large {\bf Problem 1}}

We write a Matlab code for computing
\[
 b = {\mathsf A}x,\quad x\sim{\mathcal N}(0,\sigma^2{\mathsf I}_3),
\]
where $\sigma = 0.1$. This is how I include a snippet of my Matlab code:

\begin{verbatim}
A     = [1,2,3, ...
         4,5,6];
sigma = 0.1;
x     = sigma*randn(3,1);   % Random input
b     = A*x;
\end{verbatim}
